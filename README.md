🔐 AI-Based URL Attack Detection System Using IP Analysis

📌 Overview

AI-Based URL Attack Detection System is a cybersecurity web application designed to identify malicious URLs and detect possible web attacks using URL pattern analysis and IP-based threat identification.

The system analyzes URLs entered by users and detects attacks such as SQL Injection, Cross-Site Scripting (XSS), and Path Traversal attacks.

✨ Features

🔍 URL attack detection

🛡️ Detection of SQL Injection attacks

⚠️ XSS (Cross-Site Scripting) detection

📂 Path Traversal attack detection

🌐 IP address extraction and analysis

📊 Risk level identification (Low / Medium / High)

📁 Scan history management

🔐 User login and authentication

💻 Interactive web interface
